# Instruções

- instalar as dependências exigidas:
```
    pip install -r requirements.txt
```
- Entrar na pasta:
```
    cd hospitais_cts_udds
```
- Rodar a rotina:
```
    python cnes.py
```


### Após o último comando, espera-se obter quatro arquivos na raíz desta pasta, sendo eles:
```
    etnias.csv
    hospitais.csv
    municipios.csv
    udd.csv
```
